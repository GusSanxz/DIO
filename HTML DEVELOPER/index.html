HTML

O HTML, uma linguagem de marcacao, nao de programacao, Uma linguagem de marcação serve para "marcar" ou "etiquetar" coisas e dizer como elas devem ser apresentadas. No HTML, temos a estrutura de Head, onde se botam algumas informacoes que nao aparecem diretamente na pagina, e Body, onde informa todos os elementos da tela.

TAGS

As Tags sao as "etiquetas" que se usa para poder dizer ao navegador como deve ser exibido um pedaco especifico do conteudo em uma pagina da web. Existem inumeras tags, cada uma com uma funcao diferente.

Atributos das Tags

Atriutos alteram a maneira de como o navegador pode entender o conteudo daquela tag, podendo assim criar diversas maneiras de manipular o conteudo exibido. O exemplo dado no curso foi do "id" e "style" e "src".

O primeiro da uma identificacao para aquele item, de maneira que possa ser utilizada para poder acessar aquele item, por exemplo, quando um outro chamar aquela identificacao.

O segundo chama o css para estilizar aquela marcacao, dando uma gama de alteracoes visuais. O terceiro chama um arquivo/item/imagem, seja que esteja na mesma rede ou que sejam direto da internet.

Tags de Texto

Tags de texto sao usadaas para poder diferenciar um padrao de textos. Exemplos dados no curso:

h1 ao h6: sao marcacoes de titulos e subitulos, de maneira decrescente. O navegador entende que um e um titulo e outro e um subtitulo.

p: E um texto em si, um paragrafo.

blockquote: Trata-se de uma citacao, para quando voce for querer informar uma frase que nao for sua.

Listas

Sobre listas, o curso mencionou duas, sendo elas:

ol(Ordenadas): seguem uma sequencia numerica, caso voce pretenda manter uma lista informando uma sequencia correta. Ex:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

ul(Nao ordenadas): nao seguem qualquer sequencia logica, apenas colocando um simbolo ao lado, indicando uma lista. Ela vem por padrao, caso nao identifique que tipo de lista esta sendo feita. Ex:

  • Primeiro item
  • Segundo item
  • Terceiro item

Links:

No HTML, utiliza o "a" para poder "ancorar" em algum lugar. Isso tem relacao com a atividade mencionada anteriormente, onde posso selecionar uma identificacao vinda de um "id" para poder alcancar aquele local especifico.
